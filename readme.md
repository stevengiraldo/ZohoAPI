## Zoho Invoices API Example

This is a simple interface that combine Laravel and ExtJS to connect with Zoho API. Getting invoices and showing in a grid through a ExtJS component.


### Used technologies & Services

* ExtJS
* Laravel
* Zoho API
* Heroku
